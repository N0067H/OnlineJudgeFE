## Get Started
### Linux

```bash
npm install

export NODE_ENV=development 
npm run build:dll

export TARGET=http://Your-backend

npm run dev
```
### Windows

```bash
npm install

set NODE_ENV=development
npm run build:dll

set TARGET=http://Your-backend

npm run dev
```
